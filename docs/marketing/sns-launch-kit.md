## 6. NG表現（コンプラ上ぜったい使わない）

出会い系・パパ活・風俗を想起させる表現は、非・出会い系の立ち位置を崩すため厳禁です。

| ❌ 使わない | ✅ 言い換え |
|---|---|
| 会える／出会える／出会い | 一緒に遊べる／ゲーム仲間が見つかる |
| 彼女・彼氏／恋活・婚活 | ゲーム仲間・フレンド |
| かわいい子と／女の子と（性的訴求） | 気の合う仲間と／上手い人と |
| 稼げる（過度な煽り）／即金 | 遊ぶ時間が報酬コインに（控えめに） |
| 貢ぐ／投げ銭で1位／ランキング課金 | 応援ではなく「プレイ実績で評価」 |
| デート／2人きり／会いに行く | 一緒にプレイ／オンラインで遊ぶ |

- 「稼げる」系は**過度に煽らない**（換金は弁護士確認まで開始しない方針とも整合）。
- 投げ銭・課金額でのランキングは存在しません。ランキング＝**プレイ実績・評価・信頼性**であることを明示。
- 未成年の利用はできません（18歳未満不可）。年齢を誤認させる表現もNG。
